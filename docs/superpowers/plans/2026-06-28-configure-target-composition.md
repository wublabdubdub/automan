# Configure Target Composition Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `./configure -c pg,ym-heap -o automan.yml` generate one TPC-C inventory with multiple benchmark targets from reusable fragments.

**Architecture:** Split TPC-C configure inputs into a shared base inventory and per-target fragments. `configure_main()` resolves short aliases, merges selected fragments into the base, writes a full inventory, and keeps single-target legacy names compatible without maintaining combination templates.

**Tech Stack:** Python 3.8, YAML via existing `automan_core.config`, Ansible inventory YAML, unittest.
